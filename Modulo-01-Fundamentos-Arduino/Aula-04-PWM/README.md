# Aula 04 — PWM

## Objetivo

Compreender o funcionamento do sinal PWM (Pulse Width Modulation) e a geração de diferentes frequências utilizando a função `tone()` do Arduino.

Nesta aula foi desenvolvido um projeto para demonstrar como variar a frequência de um sinal, observando os diferentes sons produzidos por um buzzer.

---

## Conteúdos estudados

- PWM (Pulse Width Modulation)
- Frequência de sinais
- Geração de ondas sonoras
- Comunicação Serial
- Função `tone()`
- Função `Serial.println()`
- Temporização com `delay()`

---

## Componentes utilizados

- Arduino Uno
- Protoboard
- Buzzer piezoelétrico
- Jumpers
- Cabo USB

---

## Projeto desenvolvido

### Gerador de Frequências

Foi desenvolvido um gerador de frequências utilizando um buzzer conectado ao Arduino.

O programa altera automaticamente a frequência do sinal emitido, variando de **0 Hz até 980 Hz**, enquanto exibe no Monitor Serial a frequência correspondente.

Essa prática permite compreender a relação entre frequência elétrica e frequência sonora produzida pelo buzzer.
