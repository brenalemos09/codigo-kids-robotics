# Aula 01 — Primeiro Contato com Arduino

## Objetivo

Nesta primeira aula foi realizado o primeiro contato com a plataforma Arduino, conhecendo seus principais componentes, configurando o ambiente de desenvolvimento e desenvolvendo o primeiro projeto prático.

---

## Conteúdos estudados

- Conhecendo a placa Arduino Uno
- Componentes básicos
- Instalação da Arduino IDE
- Estrutura de um programa Arduino
- Upload de programas
- Introdução à linguagem Arduino (C/C++)
- Projeto Semáforo

---

## Componentes utilizados

- Arduino Uno
- Protoboard
- LEDs
- Resistores
- Jumpers
- Cabo USB

---

## Conceitos aprendidos

- Entrada e saída digital
- Estrutura `setup()` e `loop()`
- Controle de LEDs
- Temporização com `delay()`
- Organização de circuitos em protoboard
