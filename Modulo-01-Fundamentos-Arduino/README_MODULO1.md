# Módulo 01 — Fundamentos do Arduino

## Sobre este módulo

Este módulo apresenta os fundamentos do desenvolvimento de sistemas embarcados utilizando a plataforma Arduino.

Durante as aulas são explorados os principais conceitos de eletrônica básica, programação em C/C++, sensores, atuadores e comunicação entre hardware e software, proporcionando uma base sólida para projetos de robótica, automação e Internet das Coisas (IoT).

Todos os projetos foram desenvolvidos utilizando a Arduino IDE e organizados de forma incremental, permitindo acompanhar a evolução do aprendizado ao longo do curso.

---

## Objetivos

- Compreender a arquitetura básica do Arduino
- Desenvolver lógica de programação aplicada ao hardware
- Trabalhar com sensores e atuadores
- Aprender comunicação serial
- Controlar displays e servomotores
- Utilizar entradas e saídas digitais e analógicas
- Desenvolver pequenos sistemas embarcados completos
