# 🐍 Snake Game - Projeto Final do Módulo 3

Este projeto é o **trabalho final do Módulo 3** do meu curso de Arduino. Apesar de o curso ser voltado para eletrônica e programação, nesta etapa foi desenvolvido um jogo em **Python** utilizando a biblioteca **Pygame**, aplicando diversos conceitos de lógica de programação.

## 📖 Sobre o projeto

O projeto consiste em uma recriação do clássico jogo **Snake**, onde o jogador controla uma cobra que deve coletar maçãs para aumentar sua pontuação e crescer de tamanho.

O jogo termina quando:

- A cobra encosta nas bordas da tela;
- A cobra colide com o próprio corpo.

## 🎮 Controles

| Tecla | Ação |
|-------|------|
| ⬆️ | Mover para cima |
| ⬇️ | Mover para baixo |
| ⬅️ | Mover para a esquerda |
| ➡️ | Mover para a direita |

## 🚀 Funcionalidades

- Movimentação da cobra
- Crescimento da cobra ao comer a maçã
- Maçãs geradas em posições aleatórias
- Sistema de pontuação
- Detecção de colisão com as bordas
- Detecção de colisão com o próprio corpo
- Tela de Game Over

## 🛠 Tecnologias utilizadas

- Python 3
- Pygame
