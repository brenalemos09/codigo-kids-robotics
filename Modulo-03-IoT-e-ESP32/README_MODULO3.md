# Módulo 03 — IoT e ESP32

## Sobre este módulo

Este módulo apresenta tecnologias modernas utilizadas no desenvolvimento de dispositivos conectados, explorando Internet das Coisas (IoT), ESP32, comunicação com bancos de dados, servidores web e integração entre hardware e software.

Ao final do módulo são desenvolvidas aplicações completas envolvendo sensores, redes, banco de dados, Python e desenvolvimento web.

---

## Objetivos

- Aprender a utilizar o ESP32
- Desenvolver aplicações IoT
- Integrar hardware e software
- Criar servidores Web
- Trabalhar com bancos de dados
- Desenvolver aplicações conectadas à Internet
