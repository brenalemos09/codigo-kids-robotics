# Aula 17 — Calculadora Interativa em Python

## Objetivo

Desenvolver uma calculadora em Python executada no terminal, permitindo ao usuário realizar operações matemáticas básicas por meio de um menu interativo.

Nesta aula foram explorados conceitos fundamentais da linguagem Python, incluindo entrada e saída de dados, estruturas condicionais, laços de repetição e interação com o usuário.

## Projeto desenvolvido

Foi desenvolvida uma calculadora interativa executada diretamente no terminal.

Ao iniciar a aplicação, o usuário informa dois números inteiros e escolhe uma das operações disponíveis através de um menu.

As operações implementadas são:

- Adição;
- Subtração;
- Divisão;
- Multiplicação;
- Encerramento da aplicação.

Após a execução de cada cálculo, o programa retorna ao menu principal, permitindo que novas operações sejam realizadas até que o usuário escolha finalizar a aplicação.

Também foi utilizada a biblioteca `time` para inserir pequenas pausas entre as mensagens, proporcionando uma interação mais agradável com o usuário.
