# Código Crianças Robótica 🤖

> Repositório com todos os projetos desenvolvidos durante o curso **Código Kids – Robótica**, abrangendo fundamentos do Arduino, Robótica Avançada e Internet das Coisas (IoT) utilizando ESP32.

---

## 📖 Sobre o projeto

Este repositório reúne todos os projetos práticos desenvolvidos ao longo do curso **Código Kids – Robótica**, organizado de forma progressiva para documentar a evolução do aprendizado em Sistemas Embarcados, Robótica e Internet das Coisas (IoT).

Ao longo de aproximadamente **60 projetos**, são abordados desde os conceitos básicos do Arduino até aplicações completas envolvendo ESP32, servidores Web, comunicação com banco de dados, Python e dispositivos conectados.

O objetivo deste repositório é servir como **portfólio técnico**, demonstrando conhecimentos em programação embarcada, eletrônica, automação e desenvolvimento de soluções utilizando hardware e software.

---

# 🎯 Objetivos do curso

Ao longo dos três módulos foram desenvolvidas competências para:

- Programação em C/C++ para Arduino
