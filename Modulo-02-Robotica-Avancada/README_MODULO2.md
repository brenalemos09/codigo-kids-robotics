# Módulo 02 — Robótica Avançada

## Sobre este módulo

Este módulo aprofunda os conhecimentos adquiridos nos fundamentos do Arduino, aplicando-os no desenvolvimento de robôs móveis e sistemas automatizados.

São abordados conceitos de controle de motores, navegação, sensores inteligentes, automação industrial e comunicação entre dispositivos, simulando aplicações reais encontradas na indústria e em projetos de robótica.

---

## Objetivos

- Desenvolver robôs móveis
- Controlar motores DC
- Trabalhar com Ponte H
- Implementar controle de velocidade utilizando PWM
- Utilizar sensores inteligentes
- Desenvolver sistemas autônomos
- Integrar diferentes dispositivos eletrônicos
